## github自用主题

[点击这里查看代码](./github.js)

-   全局黑色
-   全局Toast
-   代码块复制按钮，一键复制代码

## Teambition自用脚本

[点击这里查看代码](./teambition.js)

-   重定向teamibtion任意页面至目标页
-   跳转至目标任务看板
-   可直接定义需要跳转的任务看板标题`new`

|变量|值|
|---|---|
|targetURL|目标任务卡片页URL|
|columnTitle|columnTitle 填写你想定位的任务看板标题|
